플랜코리아 반응형 웹사이트 리뉴얼 기획서

  • 작업형태

    개인 작업

  • 사용 기술

    HTMLCSSjQuery

  • 사이트 분석 및 개선 방향

    글로벌 NGO 단체인 플랜의 한국지부 플랜코리아 사이트를 html, css, jQuery를 활용하여 리뉴얼한 프로젝트입니다.
    기존의 사이트는 이 단체의 정체성을 보여주는 부분이 모호하고 각 후원 콘텐츠를 클릭해야 후원 결제 페이지로 넘어가기 때문에 최종 결제까지 단계가 많았습니다. 그래서 플랜의 미션을 간결하게 명시하고 빠른 후원을 돕는 기능을 추가하여 사용 편의성을 높이는 것을 목표로 하였습니다.
    사용자가 사용하는 기기로, 언제 어디서든 후원이 가능하도록 반응형 웹사이트로 제작하였습니다. 그리고 현재 트렌드에 맞는 레이아웃을 짜고 세 가지 미션 콘텐츠를 보여줌으로써 플랜의 정체성을 공고히 하였습니다. 더불어 쉽고 빠르게 기부할 수 있도록 상단 메뉴에 후원 메뉴와 중간에 간단 기부 기능, 그리고 후원 퀵메뉴 버튼을 구현하였습니다.
