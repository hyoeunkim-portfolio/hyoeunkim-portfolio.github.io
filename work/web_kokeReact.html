코케 리엑트 적응형 쇼핑몰 리뉴얼 기획서

  • 작업형태

    개인 작업

  • 사용 기술

    HTMLCSSJavaScriptjQueryReact

  • 기획 방향

    리액트 라이브러리를 기반으로 쇼핑몰을 만드는 프로젝트로, 팀 작업을 했던 코케 사이트를 SPA에 맞춰서 구현하는 것이 목표였습니다.
    랜딩 페이지와 두 개의 서브 페이지를 작업했고, webpeck으로 기초 세팅을 하고 React Router로 페이지들을 연동해주었습니다. styled components를 활용하여 컴포넌트들을 스타일링하고 공통적인 버튼을 재사용했습니다.
    이 작업을 통해 기존의 사이트가 리액트 기반으로 했기 때문에 페이지들을 하나의 주제와 단순명료한 콘텐츠로 구성했음을 짐작할 수 있었습니다.
    현재 Redux로 장바구니 상태 관리 기능을 구현하고 있습니다.
